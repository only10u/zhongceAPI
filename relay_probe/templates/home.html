我们测什么、不承诺什么

  • 在线率 / 延迟：来自对 GET /v1/models 的定时请求与模型名子串是否命中，不等同于对话质量。
  • 掺水率：需多轮聊天、能力对照与更重的指纹分析；未接入前为占位。请勿将「—」误读为「已鉴定无掺水」。
  • 运行状态：由最近一次窗口内探测结果归一展示（如在线 / 无数据 / 未列此模型等）。
  • 非审计报告：结果仅供接入前技术参考，不构成对商户信誉或资金安全的保证。
  • Online & latency from GET /v1/models and ID substring hits—not chat quality.
  • Dilution needs deep checks; "—" is not a clean bill of health.
  • Status is derived from the latest windowed probes.
  • Not a formal audit—use for technical screening only.

常见问题

在线检测和排行里的数据有什么区别？

「在线检测」是你刚刚指定地址的一次即时请求结果；「排行」是本站已收录、按周期批量探测的全站时间窗口统计，更利于横向对比多家站点。

会保存我的 Key 吗？

试探测的 Key 只在当前 HTTP 请求中作为 Authorization 头转发，不写入数据库。仍建议您使用子账号或只读/测试用 Key。

和仅测连通性有什么区别？

除连通性外，我们关注模型目录是否声称提供目标线号，并分模型给出在线率与延迟分布；与照妖镜类站点一样，更偏向可复核的指标展示。

What is being tested?

We probe public /v1/models endpoints and match tracked model ID substrings—keys are not stored.
